contribute-to-wordpress-core
============================

An example of using Grunt for presentations.

1. `npm install -g grunt-cli`
1. clone the repo
1. open shell in the root of the repo
1. `npm install`
1. wait for it...
1. `grunt`
1. `cd public/`
1. `open index.html`
1. stand in awe (you don't actually *have* to stand, though)
